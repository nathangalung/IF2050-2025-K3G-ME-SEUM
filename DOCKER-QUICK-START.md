# Quick Start Commands

## Start Database
```bash
docker compose up -d
```

## Check Status
```bash
docker-compose ps
```

## Stop Database
```bash
docker-compose down
```

## Connect to Database
- **Web Admin:** http://localhost:8080
- **Direct Connection:** `psql -h localhost -p 5433 -U meseum_user -d me_seum`
